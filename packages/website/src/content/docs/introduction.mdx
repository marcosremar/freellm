**FreeLLM** is an OpenAI-compatible gateway that aggregates **6 free LLM providers** — Groq, Gemini, Mistral, Cerebras, NVIDIA NIM, and Ollama — into a single, reliable API with automatic failover, multi-key rotation, and a real-time dashboard.

![FreeLLM dashboard showing live request tracking across 6 LLM providers](/dashboard-demo.gif)

## The problem

You want to use LLMs without paying. Every major provider offers a free tier — but each one has its own SDK, its own rate limits, and its own downtime. You end up writing provider-switching logic, handling 429s, and babysitting API keys across five dashboards.

**FreeLLM fixes this in one line.** Change your base URL. Your existing OpenAI SDK code works against six providers with zero changes.

```python
client = OpenAI(base_url="http://localhost:3000/v1", api_key="unused")
```

That's the entire migration.

## What you get

- **One endpoint, any OpenAI SDK** — swap your base URL, keep your existing code
- **Automatic failover** — Groq rate-limited? Your request silently routes to Gemini, then Mistral, then Cerebras
- **Multi-key rotation** — stack `GROQ_API_KEY=k1,k2,k3` to triple your free capacity
- **Token usage tracking** — rolling 24h token counts per provider so you always know how much of your free budget is left
- **Smart meta-models** — `free-fast` for speed, `free-smart` for capability, `free` for max uptime
- **Built-in real-time dashboard** — provider health, live request log, latency, and token usage
- **Circuit breakers** — failing providers get taken out of rotation and tested for recovery automatically
- **Truly $0** — no markup, no subscription, no surprise bills
